#Java Web Development #Degree

In this assignment, students are required to develop a web application using the Java programming language and apply web development concepts to solve practical problems.

For my assignment, I chose to develop a web-based Library Management System. The system utilizes a Java-based web application integrated with an ASP.NET REST API to communicate with the database and manage library operations efficiently.

The system consists of several modules, including Dashboard, Member Management, Book Management, Loan Management, Pending Requests, and Fine Management.

One of the key features is the online book request module, which allows members to request books through the system. The request will remain in a pending state until it is reviewed and approved by the administrator. Once approved, the system automatically sends an email notification to the member, informing them that the requested book is ready for pickup.

The system also includes an automated fine calculation feature. When a member returns a book, the system checks the return date against the due date and automatically determines whether an overdue fine should be imposed.

In addition, the system incorporates QR Code technology to improve operational efficiency. Each member and book is assigned a unique QR code. Administrators can simply scan the member's QR code and the book's QR code to process borrowing transactions quickly and accurately. This feature significantly reduces manual data entry and speeds up the book borrowing process.

The project is developed using the following technologies:

* Java
* C#
* HTML
* CSS
* REST APIs
* Entity Framework Core
* SQL Server

These technologies are integrated to create a responsive, secure, and efficient library management system that improves book circulation, member services, and overall library administration.
